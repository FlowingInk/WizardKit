    Class ActionResult

    Stores intermediate results when the cacheSteps option is enabled.

    This class allows users to backtrack or reuse intermediate values by maintaining a mapping of action type names to their corresponding results. It stores these mappings in an array of key-value pairs, where the key is a string representing the action type name and the value is of any type.

    Remarks

    To utilize intermediate results, you must pass an appropriate configuration to the ActionBuilder constructor enabling the cacheSteps option.
